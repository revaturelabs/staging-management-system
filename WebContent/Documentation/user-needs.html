Agreed Goals

  • Authenticated Log-in
  • Ability to show current Available/Mapped/Confirmed resources for current batch types.
  • Ability to show forecasted resources for current batch types.
  • Ability to create new resources (Batches/Associates/Clients)
  • Option to modify/update resources.

    • User Stories

    As A Super User
    Log In
    Create Associates
    Create Batches
    Create Clients
    Normal User/Default
    Update Associates between different status levels(Available, Mapped, Confirmed)
    View current number of available associates in batch types
    View forecasted number of available associates in batch types
