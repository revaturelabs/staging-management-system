Introduction

This current iteration of the Staging Management System provides a useful resource in which a super user can create, view, and modify current Associates/Batches/Clients. Additionally, a normal user has the capability to view and modify current associates, link associates to clients, and change the associate status.

For more information, see the project proposal.

Use Cases

A super user may create a client, or batch. Once the batch has been created, associate(s) can be created/added/assigned to the batch. After an associate has been added the super user shares the same features with the normal user. When an associate is available, the person can be mapped to a client (user specified). Depending how the associate performed on an interview, they can then be moved to either "Available"(Bad) or "Confirmed"(Good).
