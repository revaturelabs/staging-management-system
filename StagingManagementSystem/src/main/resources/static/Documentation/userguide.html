  3. Normal User Role and General
    • From the main view, the buttons are populated according to total number that are presently in the specified criteria
    • Upon clicking a button, it populates a window with the list of that button's content.
    • Depending on the button clicked, each section (Available, Mapped, and Confirmed) have different actions that can be performed.
    • Upon refresh/reload via manual or automatic, the buttons will be dynamically modified to reflect the changes therein.
    • Associate Status Changes: An Associate's status will only be changed when moving between Available, Mapped, and Confirmed.
    • Status Definition:
      • Available: Associate is free to be paired with Client
      • Mapped: Associate is currently paired with a Client and is not available for other Clients currently
      • Confirmed: Associate is hired by Client; they are no longer available until they return or project cancelled.
